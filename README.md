# Flights API

API RESTful para gestión de vuelos y pasajeros hecha con TypeScript, Express.js y MongoDB.

## Como ejecutar



#### Configuración
Crear archivo `.env`
#### Ejecutar
```bash

npm docker:build 

```


## 🔗 Endpoints Principales

- `GET /api/flights` - Obtener vuelos 
- `GET /health` - Health check

## 🛠️ Tecnologías

- **TypeScript** 
- **Express.js** 
- **MongoDB**
